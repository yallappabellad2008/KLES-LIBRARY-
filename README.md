# 📚 Racing Library Management System

A modern **Library Management System** built using **React + TypeScript + Firebase** with dedicated portals for **Users, Employees, and Admins**.

---

# 🚀 Features

## 👨‍🎓 User Portal
- Browse available books
- Search books instantly
- View book details
- Responsive UI inspired by Formula 1 design
- Clean and modern interface

## 👨‍💼 Employee Portal
- Add new books
- Update existing books
- Delete books
- Manage library inventory

## 👑 Admin Portal
- Full system control
- Manage employees
- Manage users
- Monitor library activities

---

# 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| React | Frontend UI |
| TypeScript | Type Safety |
| Firebase | Backend & Database |
| Firestore | Database Storage |
| Vite | Fast Development Server |
| CSS | Styling |
